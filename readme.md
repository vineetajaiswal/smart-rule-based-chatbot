# 🤖 Smart Rule-Based Chatbot

A **Smart Rule-Based Chatbot** built using **Python** and **Streamlit**.

This chatbot responds to user queries using **predefined conditional logic** and can answer basic questions related to **Python**, **AI**, **Machine Learning**, **Data Science**, **GitHub**, and more.

---

## 🚀 Features

- Interactive chatbot interface
- Rule-based response system
- Answers basic questions about:
  - Python
  - AI
  - Machine Learning
  - Data Science
  - Streamlit
  - GitHub
- Modern Streamlit UI
- Fast and lightweight chatbot project

## 🛠️ Technologies Used

- Python
- Streamlit
